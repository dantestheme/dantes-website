Activating Theme

On Unix-like operating systems:
  1. Copy the dantes.conf into ~/.config/geany/colorschemes/. Create it if the directory doesn't exist.
  2. Choose the theme on Geany by going to the view menu and selecting Change Color Scheme...
On Windows:
  1. Copy the dantes.conf into C:\Users\YourUserName\AppData\Roaming\geany\colorschemes. Create it if the directory doesn't exist.
  2. Choose the theme on Geany by going to the view menu and selecting Change Color Scheme...
