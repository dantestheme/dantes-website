Install

  1. Start Windows Terminal and click on the down arrow symbol Ë… from menu bar. And you can see the settings menu. Or you can use Ctrl + , to open Settings directly.
  2. In the settings.json settings file for Windows Terminal, find the schemes section and paste the content of Monte-Cristo.json.

  3. Once the color scheme has been defined, it's time to enable it. Find the profiles section and add a colorScheme value to the default profile
